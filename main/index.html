Información general 📒

La aplicación esta diseñada para que los usuarios puedan practicar simulacros o test personalizables. Se puede ajustar el tiempo del temporizador a finalizar y la cantidad de preguntas que puede contener el test.

Examen aleatorio 📝

Esta opción permite realizar un test con preguntas al azar de todos los temas.

Opciones del navbar 📈

El navbar contiene únicamente dos opciones, uno para regresar la pantalla principal o de inicio y la segunda opción es un boton de hamburguesa que permite al usuario cerrar la sesión o ir a información.
